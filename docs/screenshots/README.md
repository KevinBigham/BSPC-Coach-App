# Screenshots And Demo Data

This folder is reserved for redacted product screenshots, demo walkthrough media, and reviewer-friendly visuals.

Rules:

- Do not commit screenshots with real swimmer names, family names, emails, phone numbers, attendance records, notes, invite codes, audio/video thumbnails, or meet exports.
- Use demo Firebase projects, seeded fake data, or fully redacted images.
- Prefer a small set of stable screenshots that explain the app quickly: dashboard, attendance, swimmer profile, practice plan, calendar, parent portal.
- Name files by view and date, for example `attendance-demo-2026-04.png`.

Current status: no screenshots are committed yet. This is intentional until a redacted demo data set is available.
